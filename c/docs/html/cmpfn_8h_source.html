31 /*
32  * Type: CompareFn
33  * ---------------
34  * This type defines the type space of comparison functions,
35  * each of which take the addresses of their arguments and
36  * return an integer from the set {-1, 0, +1} depending on
37  * whether the first argument is less than, equal to, or
38  * greater than the second.
39  */
40
